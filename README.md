# Book server

## Technologies
- Node.js
- MongoDB
- Express

## Dependencies
- bcrypt
- body-parser
- express
- jsonwebtoken
- mongoose
- morgan
- multer
- nodemon

## Features

- Adding books & authors into MongoDB
- Retrieving data from database

## Installation & start
First **_npm install_**

Second **_nodemon server.js_**

*To check if the GET and POST methods work you need to use Postman*

## Requirements

* Installed Node.js
* MongoDB
* Postman
